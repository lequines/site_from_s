# Internet DIMA landing

Минимальный статический сайт с Python-backend на стандартной библиотеке. Backend раздаёт `index.html`, файлы из `static/` и принимает заявки на `/order` для отправки в Telegram.

## Настройка

1. Создайте `.env` рядом с `main.py` или заполните существующий файл.
2. Укажите обязательные переменные:

```env
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=
```

3. Для запуска на VPS обычно нужно слушать внешний интерфейс:

```env
HOST=0.0.0.0
PORT=8000
```

SMTP-переменные из `.env.example` необязательны. Если они заполнены, backend дополнительно попробует отправить копию заявки на почту.

## Запуск

```bash
python -m pip install -r requirements.txt
python main.py
```

После запуска сайт будет доступен по адресу, который выведет сервер, например `http://127.0.0.1:8000`.
